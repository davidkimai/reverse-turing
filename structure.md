# Reverse-Turing Repository Structure

This document outlines the complete repository structure for the Reverse-Turing project, organizing benchmarks, traces, and analysis frameworks.

│
├── README.md                      # Core project overview and coherence hypothesis
│
├── benchmarks/                    # Standardized assessment frameworks
│   ├── academic/                  # Educational standardized tests
│   │   ├── gaokao/                # Chinese national college entrance examination

│   │   └── actuarial/             # Society of Actuaries exams
│   │
│   ├── reasoning/                 # Logic and reasoning assessments
│   │   ├── lsat/                  # Law School Admission Test
│   │   ├── gmat/                  # Graduate Management Admission Test
│   │   ├── formal_logic/          # Formal logic assessments
│   │   └── paradox_sets/          # Logical paradox resolution tasks
│   │
│   └── comprehensive/             # Interdisciplinary assessments
│       ├── ethics/                # Ethical reasoning frameworks
│       ├── philosophy/            # Philosophical problem sets
│       ├── cross_cultural/        # Culturally-situated reasoning

├── traces/                        # Recursive coherence traces
│   ├── attentional/               # Attention flow analysis
│   │   ├── attribution_maps/      # Source-to-output attribution

│   │
│   ├── conceptual/                # Concept integration measures
│   │   ├── cross_domain/          # Cross-domain concept transfer
│   │   ├── contradiction/         # Contradiction resolution traces
│   │   └── abstraction/           # Abstraction level transitions
│   │
│   └── collapse/                  # Failure mode analysis
│       ├── recursion_limits/      # Maximum recursive depth

│   │

│   ├── consistency/               # Cross-domain consistency

    └── analysis/                  # Analysis frameworks
        ├── residue_classification/ # Symbolic Residue classification tools
        ├── coherence_mapping/     # Tools for mapping coherence
        └── interpretability/      # Interpretability analysis frameworks

This structure systematically organizes both the empirical measurements (benchmarks, standardized scores) and the theoretical frameworks (traces, coherence maps) needed to comprehensively assess recursive coherence across cognitive domains.
